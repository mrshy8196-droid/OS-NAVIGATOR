Section 5: Java Threads

Summary

A Thread is an execution path and part of a process. Threads run concurrently and share memory. Threads are managed by the JVM.

Section 6: Java Multi-Threading

Summary

A Thread is the smallest unit of execution in a process. Java supports multi-threading which allows concurrent execution of tasks to improve performance and responsiveness.

What is a Thread?

A thread is a lightweight sub-process that runs independently within a program.

Ways to Create a Thread

MethodDescription
Extending Thread ClassCreate a class that extends Thread class
Implementing RunnableImplement Runnable interface

start() vs run()

start() creates a new thread while run() executes in the main thread.

Example Code

class MyRunnable implements Runnable {
  public void run() { System.out.println("Thread is running"); }
}
Thread t1 = new Thread(new MyRunnable());
t1.start();
