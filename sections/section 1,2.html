Section 1: GUI vs CLI Operations

Summary

GUI (Graphical User Interface) allows users to interact using mouse and windows, while CLI (Command Line Interface) uses text commands. Both can perform system tasks like managing processes, folders, and system information.

File stores data and has extension, folder organizes files.
