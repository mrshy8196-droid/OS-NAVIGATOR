Wildcards & Attributes ( Section 3 & Section 4 )

Section 3: Wildcards

Summary

Wildcards are symbols used to represent one or more characters. The (*) represents zero or more characters, while (?) represents exactly one character. Use 'ATTRIB' to change file properties like Hidden (+H) or Read-only (+R).

Commands Table

CommandDescription
DIR *.txtDisplays all files ending with .txt
DEL *.*Deletes all files in the current folder
COPY a* folderCopies files starting with 'a' to a folder
ATTRIB +HHides the file
ATTRIB -RRemoves the read-only attribute

Section 4: Attributes

Summary

Environment variables store information used by the OS. System variables are for all users, user variables are for one user. Batch files (.bat) automate tasks.

Commands Table

CommandDescription
setDisplays all environment variables
echo %username%Displays current user name
echo %temp%Displays temporary files path
@echo offHides commands and shows results
pauseStops execution until a key is pressed
